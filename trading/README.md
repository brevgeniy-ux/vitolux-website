# UT Bot Strategy (Pine Script v5)

Доработка индикатора **UT Bot Alerts** до полноценной торговой стратегии для TradingView.

## Файлы
- `ut-bot-strategy.pine` — готовая стратегия (Pine Script v5).

## Найденные изъяны оригинала и их исправление

| # | Проблема в оригинале (`study()` v4) | Решение в стратегии |
|---|-------------------------------------|---------------------|
| 1 | `study()` — только индикатор, нет входов/выходов, нельзя протестировать | Переписано на `strategy()` с `entry/exit` и бэктестом |
| 2 | `ema = ema(src, 1)` — EMA с периодом 1 == `src`, строка бессмысленна | Удалена; кроссовер считается напрямую по `src` |
| 3 | Нет риск-менеджмента (стоп/тейк/размер позиции) | ATR-стоп, тейк по R:R, опциональный трейлинг, размер позиции в % от капитала |
| 4 | Перерисовка на Heikin Ashi | HA сделан опциональным, `lookahead_off`, торговля по закрытию бара (`process_orders_on_close`) |
| 5 | Торговля без фильтров (флэт, контртренд) | Фильтр тренда по EMA, ограничение направления, фильтр диапазона дат |
| 6 | `//@version=4` устарел | Обновлено до v5 |
| 7 | Опечатка `"Key Vaule"`, входы без ограничений | Исправлено, добавлены `minval`/`step` |

## Важные замечания
- **Это не гарантия прибыли.** UT Bot — шумная трендследящая логика; на флэте даёт много ложных сигналов. Обязательно прогоните бэктест на своём инструменте/ТФ и оптимизируйте `Key Value`, `ATR Period`, множитель стопа и R:R.
- При включённом Heikin Ashi результаты на истории выглядят лучше, чем в реальной торговле (риск перерисовки). Для честной оценки тестируйте на обычных свечах.
- Параметры комиссии и проскальзывания в `strategy(...)` подберите под своего брокера/биржу.

## Установка
1. Откройте TradingView → Pine Editor.
2. Вставьте содержимое `ut-bot-strategy.pine`.
3. **Add to chart** → вкладка **Strategy Tester** для результатов бэктеста.
