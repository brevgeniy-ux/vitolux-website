# Admin Guide

## Вход в админ-панель

1. Откройте админ-панель: `http://yourdomain.com/admin`
2. Введите учетные данные:
   - Email: `admin@vitoluxua.com`
   - Пароль: `admin123`

⚠️ **Важно**: Измените пароль после первого входа!

## Dashboard

На главной странице админ-панели отображается:

- **Статистика заказов**: сегодня, неделя, месяц
- **Статистика продаж**: сумма продаж за периоды
- **Статистика товаров**: всего, активных, нет в наличии
- **Графики**: продажи по дням, топ товары
- **Последние заказы**: таблица последних 10 заказов

## Управление товарами

### Добавление товара

1. Перейдите в раздел "Товари"
2. Нажмите "Додати товар"
3. Заполните форму:
   - **Основное**: название (UA/EN), артикул, категория, производитель, цена, количество
   - **Описания**: краткое и полное описание (UA/EN)
   - **Характеристики**: добавьте характеристики (название + значение)
   - **Изображения**: загрузите главное и дополнительные изображения
   - **SEO**: meta title и description (UA/EN)
   - **Флаги**: популярный, новинка, скидка, опубликован
4. Нажмите "Зберегти"

### Редактирование товара

1. Найдите товар в списке
2. Нажмите "Редагувати"
3. Внесите изменения
4. Нажмите "Зберегти"

### Удаление товара

1. Найдите товар в списке
2. Нажмите "Видалити"
3. Подтвердите удаление

⚠️ **Внимание**: Удаление товара необратимо!

## Управление категориями

### Добавление категории

1. Перейдите в раздел "Категорії"
2. Нажмите "Додати категорію"
3. Заполните:
   - Название (UA/EN)
   - Родительская категория (опционально)
   - Описание
   - Изображение
   - Порядок сортировки
4. Нажмите "Зберегти"

### Изменение порядка категорий

Используйте drag & drop для изменения порядка категорий.

## Управление заказами

### Просмотр заказов

В разделе "Замовлення" отображается список всех заказов с информацией:
- Номер заказа
- Дата
- Клиент
- Телефон
- Сумма
- Статус

### Изменение статуса заказа

1. Найдите заказ
2. Выберите новый статус из выпадающего списка:
   - **Новий** - только что создан
   - **В обробці** - заказ обрабатывается
   - **Відправлено** - заказ отправлен клиенту
   - **Завершено** - заказ выполнен
   - **Скасовано** - заказ отменен
3. Статус сохраняется автоматически

### Просмотр деталей заказа

Нажмите "Деталі" для просмотра полной информации о заказе:
- Информация о клиенте
- Список товаров
- Способ доставки
- Способ оплаты
- История изменений статуса
- Примечания администратора

## FAQ

### Как добавить несколько изображений к товару?

После создания товара используйте функцию "Завантажити зображення" и выберите несколько файлов.

### Как сделать товар популярным?

При создании/редактировании товара установите флаг "Популярний".

### Как изменить порядок категорий?

Используйте drag & drop в списке категорий или измените поле "Порядок" в настройках категории.

### Как отправить уведомление клиенту об изменении статуса заказа?

Уведомления отправляются автоматически при изменении статуса заказа (если настроен email).

### Как экспортировать список заказов?

Функция экспорта будет добавлена в следующих версиях. Пока используйте копирование данных из таблицы.

## Безопасность

- Никогда не передавайте пароль администратора третьим лицам
- Используйте сложные пароли
- Регулярно проверяйте логи на подозрительную активность
- Делайте резервные копии базы данных
